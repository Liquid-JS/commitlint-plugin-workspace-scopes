## start a new package

```
$ pwd
/src/material-svelte/components

$ pnpm init @material-svelte/component
✔ Component name: · MyComponent
✔ Package name: · mycomponent

Scaffolding package in /src/material-svelte/components/mycomponent...

Done. Now run:

  cd mycomponent
  pnpm install
  pnpm dev

Don't forget to add devDependency to storybook:

  "@material-svelte/mycomponent": "workspace:*"

```
